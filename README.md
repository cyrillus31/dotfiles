# DOTFILES

## Requirements
___
- [stow](https://www.gnu.org/software/stow/manual/stow.html)


## Usage:
Backup some files
```shell
source backup.sh
```

Backup some files
```shell
source backup.sh
```

Stow everything
```shell
source run.sh
```
