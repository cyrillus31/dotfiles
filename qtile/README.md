# Qtile Configuration

Qtile window manager configuration for Fedora Wayland.

## Structure

```
qtile/.config/qtile/       → ~/.config/qtile/
qtile/.config/rofi/        → ~/.config/rofi/
```

## Setup

```bash
cd ~/dotfiles
make qtile
```

## Key Bindings

- **Alt+Return** - Terminal
- **Alt+e** - File Manager (nautilus)
- **Super+Alt** - rofi app launcher
- **Super+Space** - Switch keyboard layout
- **Alt+Ctrl+r** - Reload config
- **Alt+h/j/k/l** - Navigate windows
